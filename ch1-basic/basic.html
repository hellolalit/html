Chapter 1 - Basics of HTML.

Basic TAGS of HTML5 -

  1. The DOCTYPE

    The "DOCTYPE" neither a HTML tag nor an element. Is is used to tell the browser what version of HTML5 is used.This "DOCYPE" should be decalred at the first line of HTML5 document.
    There are more than 1 DOCTYPES such as SVG and MathML.

  2. HTML TAG

    HTML Tag is used to tell the computer that the html document is starting from this line of the document.
    And the closing of the tag tell the computer that at that line the html code has ended.

  3. HEAD TAG

    HEAD Tag is used to tell the computer that the heading content of the document is starting from this line of the document.
    It mailny includes meta tag, link tags, title tag and other important tags within it.

  4. TITLE TAG

    TITLE Tag is used to define the title of the website/webpage.

  5. LINK TAG

    LINK Tag is used to link CSS/other files to the HTML5 document.

  6. BODY TAG

    BODY Tag is used to define the body of the html document.
    And the closing of the tag tell the computer that specific line is the end of that body tag and the body if the HTML5 document ends at that specific line.

  7. SCRIPT TAG

    SCRIPT Tag is used to link Javascript files to the HTMl5 document.
