B Tag (bold tag) and Strong Tag

What is a "b tag"?

The "b" tag specifies bold text without any extra importance. In HTML, there are two types of bold tags: "b" and "strong" tags. "b" tags just make the text bold and do not specify any importance of that bold text, whereas "strong" tags do! This is made for SEO purposes so that search engines can understand the importance of bold text accordingly and do their task!


Let's take an example below:


Windows is good for daily work but has privacy and security issues. So, in order to counter that, we have Linux as an alternative!


In the above example, Windows is in bold text but does not have much importance, whereas Linux and privacy and security issues do! So, when we need to add importance to any bold text, we will use the "strong" tag instead of the "b" tag.



I Tag and EM Tag

What is the "i" tag?

The "i" tag in HTML is used to display the content in italic style. This tag is generally used to display technical terms, phrases, or important words in a different language.

What is the "em" tag?

The "em" tag is also one of the elements of HTML used in formatting texts. It is used to define emphasized text or statements.

What is the difference then?

The "i" tag is like putting something in italics just for looks, while the "em" tag is for adding real emphasis or importance to the text, indicating that it should be read with more attention. They may both look italicized, but "em" has a meaning beyond appearance.
Note: By default, the visual result is the same. The main difference between these two tags is that the "em" tag semantically emphasizes the important word or section of words, while the "i" tag is just offset text conventionally styled in italics to show an alternative mood or voice.



Mark Tag

What is the Mark Tag?

The "mark" tag in HTML is used to define the marked text. It is used to highlight part of the text in a paragraph. The "mark" tag is new in HTML 5.

Example:

Microsoft's biggest plan was to kill Linux by buying open-source software, so it bought GitHub, then introduced MS Azure, and now development is also done on Windows instead of Linux!



DEL Tag

What is the Del Tag?

The HTML "del" tag is used to mark a portion of text that has been deleted from the document. The deleted text is rendered as strike-through text by web browsers, although this property can be changed using the CSS text-decoration property. The "del" tag requires a starting and ending tag.

Example:

Microsoft's biggest plan was to kill Linux by buying open-source software, so it bought GitHub, then introduced MS Azure, and now development is also done on Windows instead of Linux! Please think Microsoft doesn't sell data but that is absolutely wrong!
