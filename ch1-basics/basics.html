Basic Structure of HTML5 -

DOCTYPE TAG

DOCTYPE Tag is used to tell the computer that the document is HTML5 document. So, we need to declare DOCTYPE tag at the 1st start line of any HTML% document.
It also indicated that it is a HTML5 document.

HTML TAG

HTML Tag is used to tell the computer that the html document is starting from this line of the document.
And the closing of the tag tell the computer that at that line the html code has ended.

HEAD TAG

HEAD Tag is used to tell the computer that the heading comtemt og the document is starting from this line of the document.
It mailny includes meta tag, link tags, title tag and other important tags within it. And the closing of the tag tell the computer that at that line the html code has ended.

TITLE TAG

TITLE Tag is used to define the title of the website/webpage.

LINK TAG

LINK Tag is used to link CSS/other files to the HTML5 document.

BODY TAG

BODY Tag is used to define the body of the html document.
And the closing of the tag tell the computer that specific line is the end of that body tag and the body if the HTML5 document ends at that specific line.

SCRIPT TAG

SCRIPT Tag is used to link Javascript to the HTMl5 document.
