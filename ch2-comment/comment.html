Ch 2 - Comments

Comments are used to write notes, comment regarding the code (example - to explain why that specific line of code exists?), inform about something or to indicate something in the code.
It is extremely important to write comments for large and complex projects as it helps to understand the project and makes it easy to maintain it. Otherwise, it becomes very difficult and frustating for anyone to understand it, maintain it and do some changes if required.
