Chapter 7 - Links in HTML5


What is Link?

Link is a reference to another document or resource. It is a way to connect web pages together. Links are the most important part of HTML5. Without it, the internet could not be created. Links in HTML5 are generally created using the "a" tag and it's attribute "href". Here's an example shown below -


<a href="https://google.com"> Google </a>


As we saw how you can create a Link in HTML5. Now let's understand what does the attribute "href" does and inside the a tag why "Google" is written?


The "href" attribute -

"href" stands for Hypertext Reference. It is used to specify the link of a web page. The value of the "href" attribute is the URL of the page you want to link to. The URL can be a web page, an image, a video, a sound file, or any other file or link.


Why "Google" was written inside "a" tag? -

"Google" was written because we wanted to redirect to google.com website. So, we have to write only those things which indicates where the URL inside "href" will redirect the user. For example, here google.com in this example above.

The "target" attrubute -

The "target" attribute specifies where to open the linked document. It controls whether to open the link in same tab, new tab or a specific frame. The value of the "target" attribute can be one of the following values:

  • _blank,
  • _self,
  • _parent,
  • _top.
